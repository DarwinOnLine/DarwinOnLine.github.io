# Images Directory

Place your images here to use them in your articles and home page.

## Usage

```markdown
![Alt text](/assets/images/your-image.jpg)
```

## Recommended formats

- **JPEG** (`.jpg`): Photos, images with gradients
- **PNG** (`.png`): Screenshots, images with transparency
- **SVG** (`.svg`): Logos, icons, diagrams
- **WebP** (`.webp`): Modern format, smaller size

## Optimization tools

- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- [ImageOptim](https://imageoptim.com/)
